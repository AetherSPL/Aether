# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 1.x | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability in AetherScan, please report it responsibly.

**Do not** open a public GitHub issue for security vulnerabilities.

Instead, contact us at: security@aetherscan.io

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and work to address the issue promptly.

## Scope

- `server.js` — authentication, session handling, API endpoints
- Percolator AMM mechanics — H-ratio and A/K calculations
- RPC endpoint security
- Agent wallet key management
